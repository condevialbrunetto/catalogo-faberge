## 1. К.ФАБЕРЖЕ with Assay Mark 88 (punzone102BIS)

### 1.1. Identification (lido:objectIdentificationWrap)

| LIDO Field | Description |
| :--- | :--- |
| **lido:lidoRecID** | `FABERGE-MARK-102` |
| **lido:category** | Silversmith's Hallmark / Maker's Mark |

### 1.2. Mark Description

- **Primary Mark**: **К.ФАБЕРЖЕ** (K.FABERGE in Cyrillic)
- **Secondary Mark**: **88** (Assay mark indicating 88 zolotniki = 916/1000 silver purity)
- **Script**: Cyrillic, rectangular cartouche
- **Period**: 1896–1908 (Pre-Kokoshnik period, St. Petersburg)

### 1.3. Iconography

![К.ФАБЕРЖЕ 88](puncoes/punzone102BIS.jpeg)

*Fig. 1: Fabergé maker's mark "К.ФАБЕРЖЕ" with 88 zolotniki assay mark. St. Petersburg, c. 1896–1908.*

### 1.4. Technical Analysis

This mark represents the **standard Fabergé firm mark** used during the late Imperial period. The "88" assay mark indicates high-grade silver (916/1000), equivalent to British sterling. This combination was used on silver objects produced at the main Fabergé workshops in St. Petersburg before the introduction of the Kokoshnik hallmarking system in 1908.

## 2. К.ФАБЕРЖЕ with Imperial Warrant (punzone109BIS)

### 2.1. Identification (lido:objectIdentificationWrap)

| LIDO Field | Description |
| :--- | :--- |
| **lido:lidoRecID** | `FABERGE-MARK-109` |
| **lido:category** | Silversmith's Hallmark with Imperial Warrant |

### 2.2. Mark Description

- **Primary Mark**: **К.ФАБЕРЖЕ** (K.FABERGE in Cyrillic)
- **Secondary Mark**: **Imperial Double-Headed Eagle** (Государственный герб)
- **Script**: Cyrillic, rectangular cartouche
- **Period**: 1885–1917 (Imperial Warrant period)

### 2.3. Iconography

![К.ФАБЕРЖЕ with Eagle](puncoes/punzone109BIS.jpeg)

*Fig. 2: Fabergé maker's mark "К.ФАБЕРЖЕ" accompanied by the Imperial Double-Headed Eagle warrant mark. This combination indicates objects produced under Imperial patronage. St. Petersburg, c. 1885–1917.*

### 2.4. Technical Analysis

The presence of the **Imperial Double-Headed Eagle** above the Fabergé name indicates that this piece was produced during the period when Fabergé held the title of **"Supplier to the Imperial Court"** (Поставщик Высочайшего Двора). This warrant was first granted in 1885 by Tsar Alexander III and continued under Nicholas II until the Revolution of 1917. Objects bearing this combination of marks are of particular significance as they confirm production during the height of Fabergé's Imperial patronage.

## 3. ФАБЕРЖЕ with Imperial Eagle (punzone110BIS)

### 3.1. Identification (lido:objectIdentificationWrap)

| LIDO Field | Description |
| :--- | :--- |
| **lido:lidoRecID** | `FABERGE-MARK-110` |
| **lido:category** | Silversmith's Hallmark with Imperial Warrant |

### 3.2. Mark Description

- **Primary Mark**: **ФАБЕРЖЕ** (FABERGE in Cyrillic, without initial)
- **Secondary Mark**: **Imperial Double-Headed Eagle** (smaller format)
- **Script**: Cyrillic, rectangular cartouche
- **Period**: 1890s–1917

### 3.3. Iconography

![ФАБЕРЖЕ with Eagle](puncoes/punzone110BIS.jpeg)

*Fig. 3: Fabergé maker's mark "ФАБЕРЖЕ" (without the initial "К") with Imperial Eagle. This variant was used on smaller objects where space was limited. St. Petersburg, c. 1890s–1917.*

### 3.4. Technical Analysis

This variant of the Fabergé mark **omits the initial "К"** (for Karl/Carl), presenting only the surname in Cyrillic. The accompanying Imperial Eagle confirms the Imperial warrant. This shorter form was typically employed on smaller objects or in confined spaces where the full "К.ФАБЕРЖЕ" mark would not fit properly. The mark remains fully authentic and carries the same provenance significance.

## 4. ФАБЕРЖЕ with Imperial Eagle (Circular) (punzone116BIS)

| **lido:lidoRecID** | `FABERGE-MARK-116` |
| **lido:category** | Silversmith's Hallmark with Imperial Warrant |

### 4.2. Mark Description

- **Primary Mark**: **ФАБЕРЖЕ** (FABERGE in Cyrillic)
- **Secondary Mark**: **Imperial Double-Headed Eagle** in circular punch
- **Script**: Cyrillic, rectangular cartouche
- **Period**: 1899–1908

### 4.3. Iconography

![ФАБЕРЖЕ with Circular Eagle](puncoes/punzone116BIS.jpeg)

*Fig. 4: Fabergé maker's mark "ФАБЕРЖЕ" with Imperial Double-Headed Eagle in a separate circular punch. This format was common after the Paris Exposition Universelle of 1900. St. Petersburg, c. 1899–1908.*

### 4.4. Technical Analysis

The **circular format** of the Imperial Eagle punch represents a distinct variant used during the period following Fabergé's triumph at the **Paris Exposition Universelle of 1900**, where the firm received the Grand Prix and Peter Carl Fabergé was awarded the Légion d'honneur. The separation of the eagle into its own circular punch allowed for greater flexibility in marking objects of varying sizes and shapes.

## 5. ФАБЕРЖЕ with Workmaster's Initials (punzone118BIS)

### 5.1. Identification (lido:objectIdentificationWrap)

| LIDO Field | Description |
| :--- | :--- |
| **lido:lidoRecID** | `FABERGE-MARK-118` |
| **lido:category** | Silversmith's Hallmark with Workmaster Mark |

### 5.2. Mark Description

- **Primary Mark**: **ФАБЕРЖЕ** (FABERGE in Cyrillic)
- **Secondary Mark**: **Imperial Double-Headed Eagle**
- **Tertiary Mark**: **Workmaster's initials** (appears to be stylised letters, possibly "ИР" or similar)
- **Script**: Cyrillic
- **Period**: 1896–1917

### 5.3. Iconography

![ФАБЕРЖЕ with Workmaster](puncoes/punzone118BIS.jpeg)

*Fig. 5: Fabergé maker's mark "ФАБЕРЖЕ" with Imperial Eagle and workmaster's initials. The presence of the workmaster mark allows attribution to a specific workshop within the Fabergé firm. St. Petersburg, c. 1896–1917.*

### 5.4. Technical Analysis

This mark combination is of **exceptional documentary value** as it includes not only the Fabergé firm mark and Imperial warrant, but also the **personal mark of the workmaster** (мастер) responsible for the piece's creation. Fabergé employed numerous workmasters, each running semi-independent workshops under the Fabergé umbrella. The most celebrated include:

- **Mikhail Perkhin** (М.П. / M.P.) – 1886–1903
- **Henrik Wigström** (Г.В. / H.W.) – 1903–1917
- **August Holmström** (А.Г. / A.H.) – jewellery
- **Erik Kollin** (Э.К. / E.K.) – gold work

The identification of the specific workmaster requires comparison with documented examples.

## 7. КФ Monogram with Kokoshnik (punzone124BIS)

### 7.1. Identification (lido:objectIdentificationWrap)

| LIDO Field | Description |
| :--- | :--- |
| **lido:lidoRecID** | `FABERGE-MARK-124` |
| **lido:category** | Silversmith's Hallmark with State Assay Mark |

### 7.2. Mark Description

- **Primary Mark**: **КФ** (K.F. monogram – Karl Fabergé)
- **Secondary Mark**: **Kokoshnik** (female head in profile facing right)
- **Script**: Cyrillic initials in rectangular cartouche
- **Period**: 1908–1917 (Kokoshnik period)

### 7.3. Iconography

![КФ with Kokoshnik](puncoes/punzone124BIS.jpeg)

*Fig. 7: Fabergé monogram "КФ" (Karl Fabergé) with Kokoshnik assay mark. The Kokoshnik (female head in traditional Russian headdress) facing right indicates St. Petersburg assay office, post-1908. St. Petersburg, c. 1908–1917.*

### 7.4. Technical Analysis

This mark combination is **diagnostic of the final period** of Fabergé production (1908–1917). The **Kokoshnik** (кокошник) hallmarking system was introduced in 1908, replacing the earlier numeric zolotniki system. Key features:

- **Kokoshnik facing right**: St. Petersburg assay office
- **Kokoshnik facing left**: Moscow assay office
- **КФ monogram**: Abbreviated form of Karl Fabergé's name

Objects bearing this mark combination can be precisely dated to the period between the introduction of the Kokoshnik system (1908) and the closure of the Fabergé workshops following the Revolution (1917).
